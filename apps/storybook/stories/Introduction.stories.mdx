# Social Blog UI Component Library

This Storybook contains UI components from all frameworks used in our monorepo:

- **React Components**: Modern components for web application
- **Vue Components**: Components for admin dashboard
- **Svelte Components**: Components for moderation interface

## Component Structure

Components are organized following Atomic Design principles:

- **Atoms**: Basic building blocks like buttons, inputs, icons
- **Molecules**: Combinations of atoms like forms, search bars, cards
- **Organisms**: Complex UI sections like headers, sidebars, content panels
- **Templates**: Page-level structures that organize organisms
- **Pages**: Specific implementations of templates

## Usage

Select components from the sidebar to view examples, documentation, and interact with them in the Canvas tab.
